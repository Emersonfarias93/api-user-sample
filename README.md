## API USER com Domain-Driven Design (DDD).
### 📋 Pré-requisitos
```
    JAVA JDK 21; 
    MAVEN 3; 
```

### 🔧 Instalação

```shell
  git clone https://github.com/Emersonfarias93/api-user-sample.git

  cd api-user-sample

  mvn clear install 

  mvn spring-boot run
```
